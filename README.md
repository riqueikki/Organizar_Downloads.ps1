# Organizar_Downloads.ps1
# 📂 Organizador Automático de Downloads

Um script em PowerShell simples e eficiente para organizar a pasta de Downloads do Windows, movendo arquivos soltos para pastas categorizadas por tipo (Imagens, Documentos, Vídeos, Músicas, Compactados, Instaladores e Outros).

## 🚀 Como usar

Você pode executar este script de duas maneiras:

### Opção 1: Copiar e colar no Terminal
1. Abra o **PowerShell**.
2. Copie o código do arquivo `Organizar_Downloads.ps1` e cole no terminal.
3. Aperte **Enter**.

### Opção 2: Executar como Script (Dois cliques)
Para rodar o script diretamente clicando com o botão direito nele, o Windows precisa de permissão:
1. Abra o **PowerShell como Administrador**.
2. Execute o comando abaixo para permitir scripts locais:
   ```powershell
   Set-ExecutionPolicy RemoteSigned
